##### Controls which items will be displayed in the Tab Context Menu

<br />

<Image class="help-img" src={tabContextMenu} alt="image" />

- Show Tab Context Menu on tabbar

  :::note[no-title]
  By default, Firefox uses the toolbar context menu for the tabbar
  :::

- Order of items in the context menu<br/>
  Choose how menu items are arranged in the tab context menu. `Tab Mix Plus order` uses TabMix's optimized arrangement, while `Firefox build-in order` preserves Firefox's default menu structure.

###### <span class="font-normal">Show in </span>Tab Context Menu

<div class="ml-3">
  Put a check mark ( &#8730; ) next to all items you want displayed in the Tab Context Menu All
  checked items will appear in the Tab Context Menu
</div>
